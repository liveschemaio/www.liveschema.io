LiveSchema

Map your service, application and infrastructure on the interactive LiveSchema web portal displaying objects and links states online and realtime.
LiveSchema is availiable as the cloud solution and for on-premises deployment.

Currently supported

  • Multilevel schemas drawing
  • One level object connectivity
  • Associated documents and properties with live access
  • State control and color show
  • 3 days state history storage
  • 3 days measurements history storage
  • Inventory database
  • Geo map placement support
  • SQL, HTTP, Ext-Script, MQTT, Config&Binary change control
  • RU, EN, IT languages support
  • RESTful API for integrations and development

Features planned

  • PDF export (soon!)
  • Multilevel and Crosslevel connection links
  • HW/SW components discovery
  • HW/SW configuration control
  • Network service and info flow traffic autodiscovery
  • Service identification and mapping
  • Elements utilisation calculation
  • VSD import and export
  • Colour themes support (light or dark background switcher)
  • Spanish translation
  • Time Machine (bomb!) - the timeline to see state of the schema in a past
  • Time Machine video export
  • Group of objects cloning
  • Link cloning
  • 3D view (bomb!)
  • VR view (bomb!)
