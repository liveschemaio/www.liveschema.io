LiveScheme

Map your service, application and infrastructure on interactive WEB portal of LiveSchema displaying objects and links states online.
LiveSchema is availiable on cloud or on premises deployment.

Currently supported:

  • Multi-level map drawing
  • Per level object connectivity
  • Discovery network elements
  • State control and color show
  • 3 days state history storage
  • 3 days measurements history storage
  • Inventory database
  • Geo map placement support
  • RU, EN, IT languages support
  • OpenAPI for integration and development

Features planned:

  • PDF export (comming soon!)
  • multi-level and cross-level connection links
  • HW/SW components discovery
  • HW/SW configuration control
  • Network service and info flow traffic auto discovery
  • Service identification and mapping
  • Elements utilisation calculation
  • VSD export
  • VSD import
  • Colour themes support (light or dark background switcher)
  • Spanish lang suport
