REFERENCES

/node/api/refs

GET

Возвращает список всех существующих Ссылок в системе в виде массива JSON-документов.

Body ответа сервера:

[
  {
    "id": <ID Ссылки>,
    "target": <ID Объекта, на который ссылается Ссылка>,
    "owner": <ID родительского Объекта Ссылки>,
    "client_data": "{
        <Параметры отображения Ссылки>
    }"
  },
  ...
]
POST

Создает новую Ссылку.

Body ответа сервера:

{
  "target": <ID Объекта, на который ссылается Ссылка>,
  "owner": <ID родительского Объекта Ссылки>,
}

/node/api/refs/:id

PATCH

Модифицирует Ссылку с указанным ID.

DELETE

Удаляет Ссылку с указанным ID.
