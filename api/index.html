LiveSchema API Doc (RUS)

Здесь приведен краткий справочник по основным методам работы с продуктом LiveSchema - HTTP RESTful API. API допускает HTTP 1.1 и HTTP 2.0, возможна работа через HTTP и HTTPS. Методы выполняются при наличии аутентифицированной сессии и прав на выполнение действий.

Объекты

Объекты - основная сущность схемы.

Объекты составляют дерево. У каждого объекта есть родитель, может быть child.

Ссылки

Методы модификации "ярлыков", иль "мягких ссылок".

Связи

Связи - возможность связать объекты на схеме.

Потоки и методы работы с ними. Потоки - дополнительный визуальный слой, позволяющий поверх схемы отобразить логический обмен и направление движенья.
